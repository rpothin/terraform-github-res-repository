# Complete Example

This example demonstrates a full configuration using all key options for this module, including template-based repository creation, merge strategy configuration, and governance settings.

> **Note**: Set `template = null` (or remove the template block) and `auto_init = true` if you want to create a repository without a template. These two options are mutually exclusive.

> **Note**: `security_and_analysis` settings require GitHub Advanced Security (GHAS) or a public repository. See the root module `_header.md` for details.
